# AgentGauntlet Dataset Seeds

Seeds are generated by `dataset/import/` scripts from three open security datasets.
Do not edit these files by hand — re-run the importers to regenerate.

## Files

| File | Source | Seeds | Description |
|------|--------|-------|-------------|
| `prompt_injection_seeds.yaml` | [AdvBench](https://github.com/llm-attacks/llm-attacks) | 520 | Direct import of AdvBench harmful-behavior goals as multi-turn attack seeds |
| `tool_usage_seeds.yaml` | [ToolBench](https://github.com/OpenBMB/ToolBench) | 21 | Tool escalation patterns from ToolBench's real-world API taxonomy |
| `memory_attack_seeds.yaml` | [MemoryAgentBench](https://github.com/HazyResearch/memory-agent-bench) | 15 | Adversarial adaptations of MemoryAgentBench's four memory task types |

## Regenerating

```bash
cd dataset/import
python import_advbench.py
python import_toolbench.py
python import_memoryagentbench.py
```
